## 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式设计（内联样式，深蓝色主题）
- **JavaScript**: 交互逻辑
- **Chart.js**: 数据可视化（CDN + 本地备份）
- **FontAwesome**: 图标库

## 页面说明

### 主页 (index.html)
项目总览页面，展示贵州省数字经济发展成果。

### 子页面
1. **智算能力提升行动** - 展示算力规模、数据中心、网络能力等
2. **数据资源建设行动** - 展示8个领域的数据资源建设成果
3. **行业大模型发展行动** - 展示AI大模型在各行业的应用
4. **智能终端产业培育行动** - 展示智算服务和智能终端产业发展
5. **数字生态优化行动** - 展示3年发展规划和历程
